# StockFlow

Sistema de gestión de stock desarrollado para administrar productos, inventario y movimientos de mercadería.

## Objetivo

StockFlow busca facilitar el control de inventario para pequeños comercios, permitiendo registrar productos, consultar existencias y gestionar movimientos.

## Funcionalidades

- Registro de productos
- Consulta de stock
- Gestión de categorías
- Control de movimientos
- Panel administrativo
